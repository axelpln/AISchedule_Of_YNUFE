# 云南财经大学强智教务导入脚本
在通用强智教务课程表导入脚本的基础上为本校当前本科教务系统进行了课表课时数、上课时间等的适配，用户导入成功后仅需设置课表开始的第一日，真正做到即开即用。

> 考虑到不同年段每学期开学时间不尽相同，故未对第一日进行适配。~~其实是我不会啦~~
